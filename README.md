# Portfolio_pexelsAPI
pexelsAPIを使用した画像検索アプリ

## アーキテクチャ
MVVM

## 使用ライブラリー
* Rxswift
* RxCocoa
* SDWebImage
* Alamofire

## 機能
* API通信
* 画像検索
* 画像リスト表示（撮影者、作品名）
* リスト画像タップ後、大きい画像表示
